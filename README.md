# Performance-module-

Premium-style **KSU Performance WebUI** with:

- Smart floating panel
- Profile modes: **Max Performance**, **Balanced**, **Power Saver**
- Toggle controls for **Spoof Engine** and **CPU/GPU Lock**
- Live meter updates in the panel

## Run locally

```bash
python3 -m http.server 4173
```

Then open: `http://localhost:4173`

> Note: This project is a front-end UI prototype and does not directly change real hardware clocks.
